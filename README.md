# 🏭 HotSix — 다이캐스팅 공정 데이터 기반 품질 예측 프로젝트

> 공정 변수 이상 패턴 감지를 통한 불량 발생 사전 경보 시스템

<br>

## 📌 프로젝트 개요

다이캐스팅(Die-casting)은 용융 금속을 금형에 고압으로 주입하여 정밀한 제품을 대량 생산하는 공정입니다.

본 프로젝트는 다이캐스팅 설비에서 매 Shot마다 자동으로 수집되는 **공정 변수(주조 압력, 금형 온도, 주입 속도 등)** 및 **센서 데이터(온도, 압력 등)** 를 분석하여, 불량 여부를 사전에 예측하는 머신러닝 기반 품질 예측 시스템을 구축합니다.

### 배경 — 왜 필요한가?

현재 다이캐스팅 현장에서는 미성형, 박리, 기공 등의 불량이 발생해도 **육안 검사에 의존**하여 다음과 같은 문제가 있습니다.

- 판정 기준의 주관성과 검사 속도 한계로 생산성 저하
- 불량 발생 원인 추적이 어려워 공정 개선 및 재발 방지 대책 부족
- 공정 데이터와 품질 검사를 효과적으로 연계하지 못해 실시간 품질 관리 미흡

이를 해결하기 위해 머신러닝 기반의 불량 자동 감지 및 조기 경보 시스템을 개발합니다.

<br>

## 🎯 분석 목표

- 다이캐스팅 공정 데이터를 활용하여 **불량 여부를 Shot 단위로 사전 예측**하는 분류 모델 개발
- 공정 변수와 불량 유형(미성형, 박리, 기공 등) 간의 관계를 분석하여 **불량 발생의 주요 원인 변수 도출**
- SHAP 분석을 통한 모델 해석 및 **공정 최적화를 위한 인사이트 제공**
- 실시간 품질 예측 체계 구축으로 **조기 경보 시스템 도입 및 불량률 감소**

## 🔎 주요 분석 요소

### 1. EDA (탐색적 데이터 분석)
- 공정 변수(`Process`)와 불량 발생 간 관계 분석
- 센서 변수(`Sensor`)의 이상치 탐색 및 처리
- Product_Type별 불량률 비교 분석

### 2. 불량 예측 모델 구축 (Classification)
- **평가 목표**: Recall ≥ 0.80 유지하면서 F1-Score 최대화
- 불량 경보 시스템 특성상 불량 누락(FN)을 최소화하는 것이 최우선
- Random Forest / XGBoost / LightGBM 비교 후 최적 모델 선택

### 3. 공정 변수 중요도 분석
- **상관분석**: Pearson 상관계수 기반 주요 변수 선별
- **SHAP 분석**: 모델 예측에 대한 변수별 기여도 해석

## ⚠️ 분석 한계 및 개선 방향

### 한계

- Shot 단위 **평균값**만 존재 → 사출 순간의 시계열 파형 데이터 부재
- 금형 마모도, 원재료 로트 등 **외부 변수 누락**
- Product_Type 통합 모델로 제품 유형별 특성 희석 가능성

### 개선 방향

- Product_Type별 분리 모델 구축
- 시계열 파형 데이터 수집 후 LSTM / 1D-CNN 적용
- 실시간 OPC UA 연동 경보 시스템 구축
